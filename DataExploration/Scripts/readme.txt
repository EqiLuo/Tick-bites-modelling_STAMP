The 'DataExploration(.ipynb)' has codes used to perform Exploratory Data Analysis (EDA) on the weather data, to be used as the input (training dataset).
The 'DataExploration_1(.ipynb)' has codes used to perform EDA on the tick bites data.
The 'true_values (.py) file' has codes to extract only the true values from the output tick bites data.
The 'ABM_Outputs(.ipynb) file' has codes to visualize the outputs of the ABM.
